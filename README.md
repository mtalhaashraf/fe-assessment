# FE-Assessment

## Installation

1. Clone this repository to your local machine:

2. Navigate to project directory:

    ```bash
    cd fe-assessment
    ```

3. Install dependencies using pnpm:

    ```bash
    npm install
    ```

## Usage

1. Start the application:

    ```bash
    npm run dev
    ```

2. The application will be served at <http://localhost:3000>
